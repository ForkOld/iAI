## 机器学习中的目标函数、损失函数、代价函数有什么区别
- Loss Function 是定义在单个样本上的，算的是一个样本的误差。
- Cost Function 是定义在整个训练集上的，是所有样本误差的平均，也就是损失函数的平均。
- Object Function（目标函数 ）定义为：Cost Function + 正则化项。
